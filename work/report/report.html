S02

Have you any previous experience of HTML, CSS and/or JavaScript?
Yes, I looked at the MDN pages and the Introductory JAvascript tutorials for other courses and managed to get a good understanding of how they all work together at a very basic level

Explain the role of HTML, CSS and JavaScript in web development.
HTML determines the structure, sort of like the skeletton of the web page, CSS determines how all these structural pieces look like ot the user, and Javascript determines how these function i.e. the actions available on the webpage, generally to make a dynamic rather than a static website.

Give a brief explanation of how the browser, the HTTP protocol and the web server interacts.
Browser is the client and makes it possible to communicate to the server by providing a platform where websites can be loaded to facilitate this interaction. The HTTP protocol are the sets of rules that both the browser and server need to follow to be able to communicate.

What is your TIL for this course section?
Good resources were provided to look up more information about these topics which i will probably use a lot more of in the future.

S04

Tell me about your previous experience on node/npm or any equal programming tools.
I had no experience with node or npm before this assignment.I had heard about it and noticed it listed in many job ads but never understood when it would be appropriate to use.

How do you feel about working with JavaScript, Node and Express?
I still feel like there is not much structure to what I code in JavaScript. Especially since I am trying to use a more functional approach rather than an object oriented. Regarding Node and Express, I like Node, the command and debugger... but Express as templating tool was really uncomfortable to learn. I wonder if other templating tools would be better. Express felt so stubborn and just unwilling to co-operate in general. Really frustrating. Hated working with it, but the result was very satisfying.

Explain how you did take on the coding assignment, did you have a plan and did it work?
At first-sight the tasks were simple. I knew what I was supposed to do and was determined to tackle tasks as they came. Grade 3 tasks were a breeze, the problems arose in grade 4. I was trying to solve it, and thought I understood what I was required to do but it turns out I had a massive knowledge hole regarding JSON responses and GET/POST requests. So much so, that after I finished the tasks for Grade 4, I continued to the Grade 5 tasks and realized I had already done those and NOT Grade 4. So back I went, re-purposed my solutions that I had done earlier and filled that knowledge hole I was so blind about. In total, Grades 3-5, it took me a total of 16hrs over the weekend, although half the time was spent researching and reading online material. I had no plan other than tackling the tasks one by one.

What grade did you aim for and was it a difficult level?
I always aim for a 5. Grade 5 was actually a lot easier than Grade 4 ...in my head. In hindsight Grade 4 was super easy and straight forward but it turns out I gneuinely didn't know what we were required to do. I naively thought that every GET/POST request generated a response, and I thought responses were always JSON. I was very mistaken.

What is your TIL for this course section?
Many, many things. How to set up a node server, and organize the modules; How to use a templating framework; Basically how to set up a Server-side dynamic webpage from scratch. ...Oh, and that stupid JSON response thing, so dumb.
